- Create a user in each of the groups.
- Generate SSH keys for the user in the admin group

## Instruction

- Submit the contents of /etc/passwd, /etc/group, /etc/sudoers

## Solutions

### Steps Taken

```sh
sudo addgroup admin
sudo addgroup support
sudo addgroup engineering
sudo adduser admin --ingroup admin
sudo adduser support --ingroup support
sudo adduser engineering --ingroup engineering
sudo usermod -aG sudo admin
su admin
ssh-keygen
cat /etc/passwd
cat /etc/group
sudo cat /etc/sudoers
```
